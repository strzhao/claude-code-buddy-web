# 色彩与交互体系设计 — Brainstorm

## 项目背景

- **当前项目**: claude-code-buddy-web — 像素猫桌面伴侣应用的皮肤商店
- **技术栈**: Next.js 16 + Tailwind CSS 4 + Upstash Redis + Vercel Blob
- **现状**: 无自定义色彩体系，全部使用 Tailwind 默认色板（blue-600, gray-*, green-*, red-*）
- **色彩参考**: stringzhao.life/colors — 苔绿品牌色 + 暖灰层级 + OKLCH 色彩空间

### 设计理念
- "温润克制有机清醒"
- 苔绿作为唯一品牌色，小面积点缀

- OKLCH 色彩空间，Tailwind CSS v4

## Q&A 决策记录

### Q1: 设计调性
**选择: 活泼像素风**
- 拥抱像素艺术美学 — 更大胆的配色、像素风 UI 元素、游戏感更强
- 理由: 皮肤商店服务于像素猫桌面应用，设计应与产品气质匹配

### Q2: 色彩应用方式
**选择: 苔绿主色+像素化**
- 苔绿仍然作为品牌主色，但在像素风美学下更大胆地使用
- 可以填充更大面积、配合像素边框/阴影
- 理由: 保持与 stringzhao.life 的品牌统一，同时适应像素风调性

### Q3: 暗色模式
**选择: 双主题（亮色+暗色）**
- 像素游戏常见深色背景，能让像素素材更突出
- 同时设计亮/暗双主题
- 理由: 完整的主题系统，提升用户体验

### Q4: /colors 页面形式
**选择: 交互式展示页**
- 色板展示 + 点击复制色值
- 主题切换预览
- 组件样式 demo（Button, Badge, Card, Input, Tab）
- 像素艺术元素展示

### Q5: 应用范围
**选择: 全局应用**
- 定义色彩体系 + CSS 变量
- 替换现有页面的 Tailwind 默认色彩
- 一步到位

## 设计方向总结

**关键词**: 像素游戏感 × 苔绿品牌色 × 双主题 × 全局统一

需要平衡：
- 苔绿色系的"温润"基因 vs 像素风的"硬朗"质感
- 参考站的"克制"用色原则 vs 更大胆的像素风面积使用
- 游戏感的趣味性 vs 工具网站的实用性
